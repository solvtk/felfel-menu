# felfel-menu

Hébergement public de l'image du menu FELFEL, servie via GitHub Pages.

- **Image directe** : https://solvtk.github.io/felfel-menu/felfel_menu.png
- **Page d'affichage** : https://solvtk.github.io/felfel-menu/

`felfel_menu.png` est **publié automatiquement** par le workflow du dépôt privé `solvtk/felfel`
(deux fois par jour). Ne pas éditer l'image à la main : elle est écrasée à chaque run.

Raison d'être : le player de signalétique (BrightSign) était bloqué par la protection
anti-bot de SiteGround sur `solvtk.ch/menu`. GitHub Pages n'a pas d'anti-bot, donc le
player accède à l'image sans CAPTCHA.
